# Input Devices Reference

## キーボード

- **通信方式**: シリアル（USART）。
- **設定**: 非同期、2400bps、8bit、パリティなし、ストップビット 1。
- **制御**: MFP 内蔵の USART レジスタ（`$E88029`〜）を通じて行う。

## マウス

- **インターフェース**: SCC (Z8530) のチャンネル B を使用。
- **データ**: トラックボールの回転パルスを SCC 経由で取得。

## ジョイスティック

- **ポート1**: `$E90001`
- **ポート2**: `$E90003`
- **制御**: I/O コントローラ経由で信号（上下左右、A/B ボタン）を読み取る。
- **Atari仕様**: 汎用的な Atari 9 ピン準拠。
