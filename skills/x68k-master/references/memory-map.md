# X68000 メモリマップ（実装向け）

> 目的: 「どの領域を、どのサイズ/粒度で触るべきか」を素早く判断する。

## 24-bit 空間の大枠

| アドレス範囲 | 領域 | 実装メモ |
| :--- | :--- | :--- |
| `$000000-$BFFFFF` | メインメモリ | 起動直後は先頭に IPL の見え方が絡むため、常駐コードは初期化後に再確認する。 |
| `$C00000-$DFFFFF` | Graphic VRAM | 画面モード/プレーン構成を前提にアクセスする。線形フレームバッファ前提は避ける。 |
| `$E00000-$E7FFFF` | Text VRAM | テキスト/プレーン描画の管理領域。 |
| `$E80000-$E81FFF` | CRTC | 解像度・表示タイミング・ラスター割り込み線。 |
| `$E82000-$E83FFF` | Video Controller / Palette | 表示優先度・表示面制御・パレット関連。 |
| `$E84000-$E85FFF` | DMAC (HD63450) | 4ch DMA。連続転送はチェーン設定を使う。 |
| `$E88000-$E89FFF` | MFP (MC68901) | タイマ/GPIP/割り込み制御。 |
| `$E8A000-$E8BFFF` | RTC | 日時管理。 |
| `$E8E000-$E8FFFF` | System Port | 機種依存の制御ビット（クロック系/ADPCM関連を含む）。 |
| `$E90000-$E91FFF` | I/O Controller | 周辺 I/O 制御レジスタ。 |
| `$E92000-$E93FFF` | YM2151 (OPM) | FM 音源。 |
| `$E94000-$E95FFF` | MSM6258 | ADPCM 音源。通常 DMAC ch3 と組み合わせる。 |
| `$E98000-$E99FFF` | SCC (Z8530) | キーボード/マウス/シリアル。 |
| `$E9A000-$E9BFFF` | PPI (8255) | ジョイスティック読み取り。 |
| `$EB0000-$EB7FFF` | Sprite registers | スプライト属性/座標。 |
| `$EB8000-$EBFFFF` | Sprite VRAM | スプライトパターン。 |
| `$ED0000-$ED3FFF` | SRAM | バッテリバックアップ領域。 |
| `$F00000-$F3FFFF` | CGROM | フォント/文字パターン。 |
| `$FE0000-$FFFFFF` | IPL-ROM | IOCS を含む起動 ROM 領域。 |

## 実装ルール（エージェント向け）

- I/O 空間（目安: `$E80000` 以降）は Supervisor モードで触る。
- 16bit/32bit アクセスは偶数アドレス境界を厳守する。
- VRAM 更新とパレット更新は V-Blank 同期を前提にする。
- 「DMA で転送できる処理」を CPU ループで書かない。
