# X68000 Memory Map

X68000 の 16MB メモリ空間の割り当て詳細は以下の通りです。

| アドレス範囲 | 内容 | 備考 |
| :--- | :--- | :--- |
| `$000000` - `$BFFFFF` | メインメモリ (最大12MB) | `$000000-$00FFFF` はリセット直後IPLのコピーが見える |
| `$C00000` - `$DFFFFF` | グラフィックVRAM (2MB) | 実際に搭載されているのは512KBだが1024x1024空間を持つ |
| `$E00000` - `$E7FFFF` | テキストVRAM (512KB) | 1024x1024ドット x 4プレーン構成 |
| `$E80000` - `$E81FFF` | CRTコントローラ | 画面解像度、同期信号等の制御 |
| `$E82000` - `$E83FFF` | ビデオコントローラ | 表示モード、プライオリティ制御 |
| `$E84000` - `$E85FFF` | DMAコントローラ | HD63450 (4チャンネル) |
| `$E88000` - `$E89FFF` | MFP (MC68901) | タイマ、GPIP、USART |
| `$E8A000` - `$E8BFFF` | RTC | リアルタイムクロック |
| `$E90000` - `$E91FFF` | システムポート | ジョイスティック、ROMスイッチ等 |
| `$E92000` - `$E93FFF` | FM音源 (YM2151) | OPM |
| `$E94000` - `$E95FFF` | ADPCM | MSM6258 |
| `$E98000` - `$E99FFF` | SCC (Z8530) | RS-232C、マウス |
| `$EB0000` - `$EB7FFF` | スプライトレジスタ | 位置、属性の定義領域 |
| `$EB8000` - `$EBFFFF` | スプライトVRAM | パターンデータ領域 |
| `$ED0000` - `$ED3FFF` | SRAM (16KB) | バッテリバックアップ領域 |
| `$F00000` - `$F3FFFF` | CGROM | キャラクタジェネレータROM (フォント) |
| `$FE0000` - `$FFFFFF` | IPL-ROM | 基本入出力 (IOCS) 収納 |
