# Input Devices Reference

## キーボード

- **通信方式**: シリアル（SCC Channel A 経由）。
- **設定**: 非同期、2400bps、8bit、パリティなし、ストップビット 1。
- **制御**: SCC (Z8530) Channel A レジスタ（`$E98001`〜）を通じて行う。

## マウス

- **インターフェース**: SCC (Z8530) のチャンネル B を使用。
- **データ**: トラックボールの回転パルスを SCC 経由で取得。

## ジョイスティック

- **ポート1**: `$E90001`
- **ポート2**: `$E90003`
- **制御**: I/O コントローラ経由で信号（上下左右、A/B ボタン）を読み取る。
- **Atari仕様**: 汎用的な Atari 9 ピン準拠。

## SCC Z8530 ($E98001〜$E98007)

- チャンネルA: キーボード / チャンネルB: マウス（シリアル通信）
- IOCSを通さず直接叩く場合は、非同期通信の初期化が必要。
- MFP 内蔵 USART（`$E88029`〜）は RS-232C シリアル通信用。

## RTC RP5C15 ($E8A001〜$E8A01F)

- 時・分・秒・日・月・年・曜日の保持。
- RAM領域があり、起動時の設定保持（SRAMとは別）に使用可能。

## PPI 8255 ($E9A001〜$E9A007)

- ジョイスティックポートの読み取り。
- Bit操作でポート1/ポート2の切り替えを行う。
