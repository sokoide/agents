# Video & Sprite System

## 表示画面構成

1. **グラフィック画面**: 16/256/65536 色モード。最大 1024x1024 ドット。
2. **テキスト画面**: 1024x1024 ドット、4 プレーン。
3. **BG画面**: スクロール可能な背景。
4. **スプライト**: 高速移動オブジェクト。

## VRAMアドレス

- **Graphic VRAM**: `$C00000` - `$DFFFFF` (2MB空間)
- **Text VRAM**: `$E00000` - `$E7FFFF` (512KB)

## CRTC (CRT Controller) — `$E80000〜`

| レジスタ | オフセット | 説明 |
| :--- | :--- | :--- |
| R00 | $E80000 | 水平総文字数 |
| R01 | $E80002 | 水平表示文字数 |
| R02 | $E80004 | 水平同期信号位置 |
| R04 | $E80008 | 垂直総ラスタ数 |
| R06 | $E8000C | 垂直表示ラスタ数 |
| R09 | $E80012 | ラスター割り込みライン番号 |
| R20 | $E80028 | 画面モード (解像度・色数の切替) |

### 画面モード (R20)

| ビット | 内容 |
| :--- | :--- |
| 1-0 | 解像度: 00=1024x768, 01=512x512, 10=512x512(高解像度) |
| 3 | 65536色モード |

## パレット — `$E82000〜`

- 16色モード: 16エントリ (#0〜#15)、各エントリ 4 bytes (下位12bit有効)
- 256色モード: 256エントリ、各エントリ 4 bytes
- 65536色モード: パレット不使用 (ピクセル直接指定)

### パレットフォーマット (1エントリ = Long Word)

| ビット | 内容 |
| :--- | :--- |
| 11-8 | 青 (4bit) |
| 7-4 | 緑 (4bit) |
| 3-0 | 赤 (4bit) |

※書き込みはV-Blank期間中に行うこと（雪ノイズ防止）。

## ビデオコントローラ — `$E82000〜`

- `$E82028`: 画面モード制御レジスタ (テキスト/グラフィック表示on/off、BG画面等)
- `$E82082`: H-Scroll レジスタ
- `$E82086`: V-Scroll レジスタ

## スプライト仕様

- **表示数**: 最大 128 個。
- **サイズ**: 16x16 ドット。
- **色数**: 16 色 / スプライト。
- **レジスタ ($EB0000〜)**: X 座標、Y 座標、パターン番号、属性を指定。
- **スプライトVRAM ($EB8000〜)**: ドットパターンデータを格納。

### スプライトレジスタ構成 (8 bytes/sprite, $EB0000〜)

| オフセット | 内容 | 説明 |
| :--- | :--- | :--- |
| +0,1 | X座標 | 画面左端からのドット数 |
| +2,3 | Y座標 | 画面上端からのドット数 |
| +4 | パターン番号 | $EB8000〜のパターンインデックス |
| +5 | 属性 | プライオリティ、パレット番号等 |

## CGROM (フォント)

- **英数字**: 8x8, 8x16, 12x12, 12x24 ドット。
- **漢字**: 16x16, 24x24 ドット（JIS 第 1・第 2 水準）。
