# Storage Controllers

Inside X68000 の FDD / SASI / HDC まわりの章を要約した補助 reference。

## FDD

- FDD は独立章があり、仕様、関連ポート、FDC コマンド、フェーズ遷移が分かれている。
- X68000 の FDD 制御は FDC 本体だけでなく、I/O コントローラや OPM 側の関連ポートも絡む。
- 実装時は READY、割り込み許可、フェーズ遷移をまとめて扱う。

## HDC / SASI

- HDC/SASI も独立章がある。
- 大容量ストレージ制御は DMAC `CH1` 前提で考える。
- FDD 実装の延長で同じレジスタ設計だと決め打ちしない。

## レビュー観点

- FDD/HDC と DMAC チャネル対応が合っているか
- ポートアクセスだけでなくフェーズ遷移・状態遷移まで扱っているか
- ポーリング前提でタイムアウトやエラー系を落としていないか
