# DMAC (HD63450)

Inside X68000 の DMA 章に基づく要約。

## 概要

- 採用 DMAC は HD63450。
- 4 チャネル構成。
- X68000 では高速転送用途として FDD / HDC / ADPCM で利用される。

## チャネル割り付け

| チャネル | 用途 |
| :--- | :--- |
| `CH0` | FDD |
| `CH1` | HDC |
| `CH2` | 拡張スロット |
| `CH3` | ADPCM |

## チャネルベース

各チャネルは `0x40` バイト間隔で並ぶ。

| チャネル | ベース |
| :--- | :--- |
| `CH0` | `$E84000` |
| `CH1` | `$E84040` |
| `CH2` | `$E84080` |
| `CH3` | `$E840C0` |

## 主なレジスタオフセット

Inside X68000 のレジスタ一覧図で確認できる主要オフセット。

| オフセット | レジスタ |
| :--- | :--- |
| `+00` | `CSR` |
| `+01` | `CER` |
| `+04` | `DCR` |
| `+05` | `OCR` |
| `+06` | `SCR` |
| `+07` | `CCR` |
| `+0A` | `MTC` |
| `+0C` | `MAR` |
| `+14` | `DAR` |
| `+1A` | `BTC` |
| `+1C` | `BAR` |
| `+25` | `NIV` |
| `+27` | `EIV` |
| `+29` | `MFC` |
| `+2D` | `CPR` |
| `+31` | `DFC` |
| `+39` | `BFC` |
| `+3F` | `GCR` (`CH3` のみ) |

## 実務ルール

- DMAC 設定は、チャネル用途の固定割り付けを前提に考える。
- `MAR` と `DAR` は 32bit アドレス。整列と転送幅を必ず合わせる。
- FDD/HDC/ADPCM は CPU ループより DMAC を優先する。
- 細かい動作モードやチェーン記述子形式は、実装時に原典を再確認する。

## レビュー観点

- チャネル番号が用途と一致しているか
- `CH2` を汎用転送と誤認して既存用途と衝突していないか
- ベースアドレスとオフセットを合算するときに `0x40` 刻みを崩していないか
- `CSR/CER` の扱いを雑にして完了判定とエラー判定を混同していないか
