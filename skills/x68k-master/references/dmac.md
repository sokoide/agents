# DMAC (HD63450) Reference

## チャンネル割り当て

- **CH#0**: フロッピーディスク (FDC)
- **CH#1**: ハードディスク (HDC)
- **CH#2**: 拡張スロット (メモリ転送等に使用可能)
- **CH#3**: ADPCM

## 主要レジスタ (1チャンネルあたり)

各チャンネルのベースアドレス `$E84000 + (n * $40)` からのオフセット：

- `+00`: **CSR** (チャンネルステータス) / **CER** (エラーレジスタ)
- `+04`: **DCR** (デバイスコントロール) / **OCR** (オペレーションコントロール)
- `+06`: **SCR** (シーケンスコントロール) / **CCR** (チャンネルコントロール)
- `+0A`: **MTC** (メモリトランスファカウンタ)
- `+0C`: **MAR** (メモリアドレスレジスタ)
- `+14`: **DAR** (デバイスアドレスレジスタ)
- `+1A`: **BTC** (ベーストランスファカウンタ)
- `+1C`: **BAR** (ベースアドレスレジスタ)

## 転送モード

- **デュアルアドレスモード**: X68000 の全チャンネルで使用。
- **チェインモード**:
  - **アレイチェイン**: 連続した転送情報テーブルを参照。
  - **リンクアレイチェイン**: 各テーブルが次へのポインタを持つ。

## Human68k初期設定値例

- **OCR**: `$B2` (CH#0,1), `$32` (CH#3) - 外部要求、サイクルスチール、8bit。
- **SCR**: `$04` (CH#0,1,3) - メモリアドレス増加、デバイスアドレス固定。
