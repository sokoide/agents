# User I/O / SRAM / CGROM / IPL-ROM

メモリマップ図のうち、既存 reference で抜けていた領域の整理。

## User I/O

- `$EC0000-$ECFFFF`
- 拡張用の 64KB 領域
- システム I/O と同様、直叩きコードでは機種差と拡張カード差を前提にする

## SRAM

- `$ED0000-$ED3FFF`
- 16KB のバッテリバックアップ SRAM
- 設定保持用途として使われるため、破壊的な書き込みは慎重に扱う

## CGROM

- `$F00000-$FBFFFF`
- 文字パターン ROM
- 英数字だけでなく漢字パターン参照も含む

## IPL-ROM

- `$FE0000-$FFFFFF`
- 起動直後の IPL イメージと通常時の見え方を混同しない
- IOCS を ROM から直接使う時代の前提と、RAM 展開後の利用形態を区別する
