# X68000 ジョイスティック

2 系統（ポート#1/#2）のジョイスティック端子。アタリ規格準拠 D-Sub 9 ピン。初代〜XVI まで完全互換。

## データレジスタ

| ポート | アドレス |
| --- | --- |
| ポート#1 データ | `$E9A001` |
| ポート#2 データ | `$E9A003` |

**負論理**: スイッチ ON で '0'。

| ビット | 入力 |
| --- | --- |
| Bit 0 | 上 (UP) |
| Bit 1 | 下 (DOWN) |
| Bit 2 | 左 (LEFT) |
| Bit 3 | 右 (RIGHT) |
| Bit 4 | ボタンA (TRIGGER 1) |
| Bit 5 | ボタンB (TRIGGER 2) |

## コントロールレジスタ（`$E9A005`）

ポートのピン出力を制御。コントロールワード送出で信号線の役割を動的に変更するマルチプレクス機能を提供。多ボタンパッドやトラックマウス等の拡張デバイスは、このコントロールワードによるサンプリングが必要。

## タイミング制約

コントロールワード書き込み直後は**信号が電気的に安定していない**。数マイクロ秒のウェイト（NOP 命令等）が必要。このウェイトを無視したコードは XVI 等の高速機で深刻な誤動作を引き起こす。

## 実装例

```c
volatile unsigned char *joy1 = (unsigned char *)0xE9A001;
unsigned char raw = *joy1;
unsigned char active = ~raw;  // 負論理を正論理に反転

if (active & 0x20) { /* ボタンA */ }
if (active & 0x01) { /* 上 */ }
if (active & 0x02) { /* 下 */ }
```

## チャタリング対策

16MHz CPU はスイッチの物理的接点ノイズ（チャタリング）を数回〜数十回サンプリングする。前回値との比較で一定期間の安定を確認すること。

上/下の同時入力等の異常入力に対しても、「上優先」等の論理規定をプログラム側で設けること。
