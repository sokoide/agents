# X68000 割り込みシステム

## 割り込みレベルとデバイス割り当て

| レベル | デバイス | 主な機能 |
| --- | --- | --- |
| **7** | **NMI** | 本体背面 NMI スイッチ（オートベクタ $1F） |
| **6** | **MFP (MC68901)** | CRTC、FM音源、タイマ、キーボード |
| **5** | **SCC (Z8530)** | RS-232C、マウス |
| **4** | 拡張スロット | ユーザー増設 |
| **3** | **DMAC (HD63450)** | ADPCM、FDD、HDD 転送終了 |
| **2** | 拡張スロット | ユーザー増設 |
| **1** | I/O コントローラ | FDC、FDD、HD、プリンタ |

- SR の I0-I2（3 ビット割り込みマスク）で最低受付レベルを定義。高レベルは低レベルをプリエンプト可能。
- レベル 7（NMI）のみマスク不可。常に受付。
- **ベクタ $0F**（非初期化割り込み）: DMAC 各 ch 等の初期ベクタとして設定され、未初期化デバイスからの暴走防止に機能。

## 応答サイクル

1. 周辺デバイス → プライオリティエンコーダ → `IPL0`〜`IPL2` → CPU
2. CPU は命令境界でのみサンプリング
3. CPU → `FC0-FC2` = H（IACK）、`A1-A3` に割り込みレベル出力
4. ベクタ取得:
   - **ベクタ割り込み**: デバイスがデータバスに固有ベクタ番号を送出 → `DTACK`
   - **オートベクタ**: デバイスが `DTACK` の代わりに `VPA` を返す → レベル対応デフォルトベクタ（$18-$1F）
   - **X68000 では NMI（レベル7）のみオートベクタ（$1F）**。他は全て LSI 内部レジスタ設定のベクタ。

## 例外ベクタテーブル

| ベクタ | 役割 | 備考 |
| --- | --- | --- |
| $00-$0F | モトローラ予約 | リセット、バスエラー、アドレスエラー、$0F（非初期化）等 |
| $18-$1F | オートベクタ | $1F は NMI |
| $20-$27 | TRAP 0〜7 | $27 は DB.X ブレークポイント等 |
| $2F | IOCSコール | Human 68K 標準入出力 |
| $40-$4F | MFP | タイマ、キーボード、FM音源等 |
| $50-$5F | SCC | RS-232C、マウス |
| $60-$63 | I/Oコントローラ | $60:FDC, $61:FDD, $62:HD, $63:プリンタ |
| $64-$6B | DMAC | CH 0〜3 |

## 主要LSI ベクタ設定ポート

| LSI | ポート | Human68K 初期値 |
| --- | --- | --- |
| MFP | `$E88017` (VR) | `$40` |
| SCC | `$E98003` / `$E98007` (WR2) | `$50` |
| DMAC CH#0 | `$E84025` (NIV) | `$0F`（非初期化） |
| I/Oコントローラ | `$E9C003` | $60〜$63 を要因別に |

## MFP 割り込み制御の S ビット

VR レジスタのビット 3（S ビット）で EOI モードを選択:

- **S=0（オートEOI）**: CPU がベクタを読み取った瞬間に ISR 自動クリア。同一要因の再入に注意。
- **S=1（ソフトウェアEOI）**: 手動で ISR をクリアするまで同一要因をブロック。多重割り込みの深さを厳密制御可能。

**警告**: タイマ B（TBCR）は**キーボード通信用ボーレートジェネレータとして固定**。変更するとキーボード入力が即座に不能になる。

## ISR 実装の鉄則

- 使用する全レジスタを `MOVEM` 等でスタック退避、終了時に復帰。
- 最後は必ず `RTE` 命令で終了（例外スタックフレーム破棄＋SR 復帰）。
